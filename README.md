# valvecontrol
Valve control with ESP-32

This sketch controls a ball valve using an ESP32, enabling full or partial opening and closing.
It is based on the specifications of a 12V ball valve equipped with open/closed sensors.
A 12V power supply can be used for the relay and valve, while a step-down converter (LM2596 or equivalent) can be used to power the ESP32 (which operates at 3.5V).

The firmware for ESP32 is the PressControl.ino.merge.bin
Is possible to flash your ESp in https://maxbox.net.br/flash/flash.html
